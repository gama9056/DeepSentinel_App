# DeepSentinel_App
DeepSentinel es una solución avanzada para acceder a datos geoespaciales de alta calidad sin programar. Mediante IA en Python, transforma imágenes Sentinel de 10m a 2.5m de resolución. Solo sube un KML y elige la fecha para descargar la versión original, la mejorada o ambas. Obtén claridad superior con total simplicidad técnica.
